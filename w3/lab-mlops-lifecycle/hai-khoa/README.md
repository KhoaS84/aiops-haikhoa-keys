# MLOps Lifecycle Pipeline (hai-khoa)

Để chạy pipeline từ đầu đến cuối: Khởi động stack Docker thông qua lệnh `docker compose -f configs/docker-compose.yml up -d` (hoặc chạy file shell script `bash scripts/start_stack.sh`). Sau đó, thiết lập môi trường ảo Python bằng `uv venv` và cài đặt các thư viện cần thiết với `uv pip install --link-mode=copy 'mlflow==2.13.2' 'evidently==0.4.40' 'setuptools<70' scikit-learn pandas numpy fastapi uvicorn prometheus_client requests`. Tiếp theo, chạy lệnh huấn luyện và đăng ký mô hình v1: `$env:MLFLOW_TRACKING_URI="http://localhost:5000"; uv run python hai-khoa/pipeline.py --data data/baseline.csv`. Tiếp đến, khởi chạy API server bằng lệnh: `$env:MLFLOW_TRACKING_URI="http://localhost:5000"; uv run python hai-khoa/serve.py`. Cuối cùng, kích hoạt pipeline tự động phát hiện drift, tái huấn luyện và giám sát rollback tự động thông qua lệnh: `$env:MLFLOW_TRACKING_URI="http://localhost:5000"; uv run python hai-khoa/retrain.py --reference data/baseline.csv --current data/drifted.csv --holdout data/holdout.csv --post-deploy-eval data/post_deploy_eval.csv --serve-url http://localhost:8000 --auto-approve`.
